## Project Overview
This project analyzes an employee dataset using Microsoft Excel and Python. 
Excel was used for exploratory analysis and pivot tables, while Python (pandas)
was used for data cleaning, validation, and HR KPI calculations.

- Week-3-Employee-Data_CLEANED.csv (cleaned dataset)
- week3_cleaning.ipynb (Python cleaning and KPI calculations)
- Excel workbook with pivot tables and charts
